What's New in Astro v3.1.0

Released on May 10, 2025 - Our most significant update yet

Core Model

ModelA 8-Pro

Our most advanced language model yet, featuring enhanced reasoning capabilities comparable to LLaMa 3.3 70B, with improved context handling.

  • Significantly better at code generation and debugging
  • Knowledge cut-off date of October 2023
  • Enhanced context handling for longer conversations
  • Reduced hallucinations compared to ModelA 7
New Feature

Canvas: Interactive Editing

Edit content directly within Astro's responses using the new interactive canvas feature.

  • Use the && command to open any text in the editing canvas
  • Live formatting and syntax highlighting for code
  • Multiple formatting options and export capabilities
  • Collaborative editing when using shared workspaces
New Feature

Conversation Modes

Six specialized modes to optimize Astro for specific tasks and workflows, introduced for the first time in v3.1.0.

  • General/Basic: Standard comprehensive assistance
  • Code Assist: Optimized for coding tasks with improved debugging
  • Writing Assist: Enhanced content creation and editing
  • Analysis Assist: Better data analysis and interpretation
  • Creative Assist: Enhanced storytelling and creative projects
  • Astro Search: Nova Search powered information retrieval
Experimental

Nova Search

Our new experimental search functionality connects Astro to real-time web information.

  • Real-time information without leaving the conversation
  • Dynamic loading of search results
  • Citation tracking with source verification
  • Privacy-first design with no search history stored on our servers
Improvement

Increased Capacity

More resources available to users for their daily Astro interactions.

  • Daily token limit increased from 6K to 9K
  • Better response handling for complex queries
  • Improved conversation management
  • Enhanced overall system reliability
Performance

System-wide Optimizations

Major performance improvements across the entire Nova Suite platform.

  • 50% faster startup time
  • Reduced memory usage by 30%
  • Optimized caching for frequently used prompts
  • Better offline capabilities with enhanced local processing
